### 完整版提示词（推荐）
```
上面是一个基金经理(003501.OF)2020年底以来每个季度的重仓股,和月度净值。
根据重仓股和净值,你能否总结出来该基金经理投资框架、投资特点、

这位基金经理在净值波动（注意是波动）水平相近，风格相近的人中，
投资能力是什么情况，适合什么环境，不适合什么环境。
怎么投资这位基金经理，如何买卖？
```

## 单产品深度分析流程

### 数据准备清单
- [ ] 基金代码
- [ ] 2020年底以来季度前十大重仓股（每季度）
- [ ] 月度净值数据
- [ ] 基金经理任职时间
- [ ] 基金规模变化

### 分析步骤

#### 第一步：投资框架分析
**提示词：**
```
上面是基金[基金代码]从2020年底以来每个季度的重仓股和月度净值数据。

请分析：
1. 该基金经理的投资框架是什么？（用一句话概括）
2. 投资特点和风格特征？
3. 重点关注的指标有哪些？
4. 调仓逻辑和持股周期？
5. 重仓行业的配置逻辑？
```

**预期输出：**
- 一句话投资框架总结
- 投资风格标签（价值/成长、大盘/小盘）
- 调仓频率特征

#### 第二步：业绩归因与能力圈分析
**提示词：**
```
基于同样的数据，请深度分析：

1. 业绩归因：
   - 收益主要来自选股Alpha还是行业配置？
   - 选股Alpha的离散度如何？
   - 是否有持续稳定的超额收益来源？

2. 能力圈分析：
   - 能力圈强势领域（哪些行业/风格能持续做出超额）
   - 能力圈弱势领域（哪些行业持续毁灭价值）
   - 是否存在"能力陷阱"（自以为擅长但实际表现差的领域）

3. 和同风格、同波动水平的基金经理相比，投资能力如何？
```

**关键洞察示例：**
- "该基金经理最重仓的板块，不是他的阿尔法来源"
- "他最喜欢的行业，没有做出超额"
- "选股Alpha呈现较大离散度，整体超额不明显，收益更多来自行业配置"

#### 第三步：投资建议
**提示词：**
```
基于以上分析，请给出具体的投资建议：

1. 配置建议：
   - 适合什么市场环境？（牛市/熊市/震荡市、价值/成长风格）
   - 不适合什么环境？
   - 配置比例建议

2. 买卖策略：
   - 买入时机信号
   - 持有周期建议
   - 卖出/赎回信号

3. 风险控制：
   - 最大回撤预期
   - 需要警惕的风险点

4. 对基金经理的建议（如适用）：
   - 需要扩展哪些能力？
   - 需要保留哪些能力？
   - 应该减少哪些领域的主动偏离？
```

## 赛道产品对比分析

### 对比维度
1. **投资风格**：价值/成长、大盘/小盘
2. **选股逻辑**：自下而上/自上而下、质量/景气度
3. **换手率**：持股周期、调仓频率
4. **业绩归因**：alpha来源（选股vs行业）、beta暴露
5. **风险控制**：最大回撤、波动率控制
6. **能力圈**：各自擅长/弱势的领域

### 提示词模板
```
以下是两位基金经理的重仓股和净值数据：

基金经理A：[数据]
基金经理B：[数据]

请对比分析：
1. 两人的投资风格差异
2. 选股逻辑有什么不同
3. 业绩归因对比（谁的Alpha更稳定、主要来自选股还是行业配置）
4. 各自的优劣势和能力圈
5. 适合什么样的投资者
6. 在当前市场环境下，更看好谁？为什么？
7. 如果是你，会怎么配置这两位经理？比例如何？
```

## Smartbeta产品分析

### 分析要点
1. **指数编制方法**：加权方式、选股规则
2. **因子暴露**：价值/成长/质量/动量等因子
3. **行业分布**：集中度、分散度
4. **历史表现**：不同市场环境下的表现
5. **当前估值**：PE、PB分位数

### 提示词模板
```
以下是[指数名称]的编制规则和成分股数据：

请分析：
1. 该指数的编制逻辑和特点
2. 主要暴露在哪些因子上
3. 行业分布特征
4. 适合什么市场环境（牛市/熊市/震荡市）
5. 不适合什么环境
6. 当前估值水平如何（PE/PB分位数）
7. 投资建议（适合作为什么类型的配置）
```

## AI分析的优势

### 1. 客观犀利
- 不会因为人情关系手下留情
- 敢指出"能力陷阱"（最重仓的板块不是阿尔法来源）
- 能发现"自以为的能力圈其实是能力陷阱"

### 2. 精准归因
- 可识别选股Alpha离散度
- 能区分行业配置贡献vs个股选择贡献
- 可定位持续毁灭价值的领域

### 3. 效率极高
- 过去研究员一天的工作量，AI十分钟完成
- 可读完全市场基金报告，快速学习
- 可处理大量历史数据发现规律

### 4. 持续进化
- 模型可读最新季报持续更新认知
- 可追溯历史上相似市场环境的表现

## 输出报告结构

### 1. 执行摘要
- 一句话投资框架
- 核心结论（买/不买/观望）
- 适合的投资者类型

### 2. 基金经理画像
- 投资风格标签
- 能力圈地图（强势/弱势领域）
- 关键特征（换手率、集中度、行业偏好）

### 3. 业绩归因
- 收益来源分解（选股vs行业）

- 同风格排名

### 4. 深度洞察
- 犀利发现（如"能力陷阱"）
- 与市场认知的偏差
- 风险点提示

### 5. 投资建议
- 配置比例
- 买入时机
- 卖出信号
- 风险提示

### 6. 跟踪要点
- 需要关注的变化
- 调仓信号
- 退出条件

## 使用建议

### 1. 分析频率
- **日常跟踪**：季度报告发布后1周内
- **深度分析**：半年度/年度，或大幅调仓后
- **对比分析**：同类基金排名变化时

### 2. 适用场景
- 基金入库筛选
- 持仓基金定期检视
- 新基金经理尽调
- 组合再平衡决策

### 3. 注意事项
- AI分析基于历史数据，不代表未来表现
- 需结合定性信息（如基金经理变动、策略调整）
- 警惕AI幻觉，关键结论需交叉验证

### 4. AI的局限（不可替代的部分）
- 非结构化信息（"经理刚分手状态不好"）
- 人际网络和行业直觉
- 情绪管理和行为干预（投顾端）
