# 基金产品库入库标准确认
## 最终版（2026-03-08）

---

## 一、基金分类（8个类别分别筛选）

| 类别 | 代码示例 | 说明 |
|:---|:---|:---|
| 1. 主动股票型 | 股票仓位>80% | 纯股票投资，高波动 |
| 2. 混合型-偏股 | 股票仓位60-80% | 股债平衡，偏进攻 |
| 3. 混合型-平衡 | 股票仓位40-60% | 股债均衡 |
| 4. 混合型-偏债 | 股票仓位<40% | 债券为主，偏防御 |
| 5. FOF | 基金中基金 | 投资基金，分散度高 |
| 6. 指数增强型 | 跟踪指数+主动增强 | Beta+Alpha结合 |
| 7. 债券型-主动 | 主动债券管理 | 固收类，低波动 |
| 8. QDII | 海外投资基金 | 跨境投资 |

**关键原则：每个类别单独筛选前10%，不混排**

## 二、入库标准（三级筛选）

### 一级指标（必须全部满足，硬性门槛）

| 指标 | 标准 | 说明 |
|:---|:---|:---|
| ✅ 同类排名 | 近1年收益排名前10%（同类别） | 核心指标，必须满足 |
| ✅ 持续性 | 近6个月排名保持前20%（同类别） | 排除短期运气 |
| ✅ 基金经理 | 任职时间>1年 | 确保策略稳定性 |

**不满足任何一项，直接排除**

---

### 二级指标（质量评分，综合评估）

| 指标 | 优秀(90-100分) | 良好(70-89分) | 合格(60-69分) | 不合格(<60分) |
|:---|:---|:---|:---|:---|
| 夏普比率 | >同类别75%分位 | >同类别中位数 | >同类别25%分位 | <同类别25%分位 |
| 最大回撤 | <同类别25%分位 | <同类别中位数 | <同类别75%分位 | >同类别75%分位 |
| 年化波动率 | <同类别25%分位 | <同类别中位数 | <同类别75%分位 | >同类别75%分位 |
| 月度胜率 | >70% | >60% | >50% | <50% |
| 盈亏比 | >2.0 | >1.5 | >1.0 | <1.0 |

**评分规则：**
- 总分=夏普30% + 回撤25% + 波动率20% + 胜率15% + 盈亏比10%
- 入库要求：总分≥70分（良好及以上）

---

### 三级指标（深度评估，入库后分析）

| 指标 | 用途 | 数据来源 |
|:---|:---|:---|
| 换手率 | 评估调仓频率 | 半年报/年报 |
| 行业集中度 | 评估风险分散度 | 季报持仓 |
| 重仓股稳定性 | 评估持股周期 | 季报持仓对比 |
| 规模变化 | 评估容量影响 | 定期公告 |
| 机构持有比例 | 评估专业认可度 | 半年报/年报 |

**三级指标不入库门槛，但影响深度研究优先级**

---

### 一票否决项（出现即不入库）

| 情况 | 说明 |
|:---|:---|
| ❌ 基金经理变更 | 现任经理任职<6个月 |
| ❌ 巨额赎回 | 单日净值异常波动>5% |
| ❌ 违规记录 | 监管处罚、信息披露问题 |
| ❌ 规模过小 | <1亿元，清盘风险 |
| ❌ 波动率过高 | >同类别90%分位（极端激进）|
| ❌ 数据异常 | 净值数据缺失>20% |

## 三、入库流程漏斗

```
全市场基金 ~4,900只
    ↓ 按8个类别分别筛选前10%
候选基金 ~490只
    ↓ 一级指标筛选（3项必须全部满足）
一级通过 ~50-80只
    ↓ 二级指标评分（总分≥70分）
二级通过 ~20-30只
    ↓ 一票否决检查（6项检查）
最终候选 ~15-20只
    ↓ 快速分析（月度）
月度分析 ~8-10只
    ↓ 深度研究+入库（季度）
季度入库 ~5-8只（核心库，分类型）
```

## 四、产品库分级

| 级别 | 标准 | 数量目标 | 分析深度 | 更新频率 |
|:---|:---|---:|:---|:---:|
| **核心库** | 连续2个季度前10% + 质量评分≥80分 | 50-80只 | 深度（9维度） | 季度跟踪 |
| **观察库** | 首次进入前10% 或 质量评分70-80分 | 100-150只 | 标准（快速分析） | 月度跟踪 |
| **备选库** | 曾进入前10%但目前退出 | 200-300只 | 快速（监控） | 双周监控 |

---

## 五、分析区间权重

| 分析区间 | 权重 | 用途 |
|:---|:---:|:---|
| 近6个月 | 30% | 短期动量、最新调仓效果 |
| 近1年 | 40% | 核心考核区间、夏普比率有效 |
| 近2年 | 15% | 中期能力、穿越小周期 |
| 近3年 | 10% | 长期能力、策略可持续性 |
| 成立以来 | 5% | 历史业绩参考 |

## 六、更新频率

| 任务 | 频率 | 内容 | 工作量 |
|:---|:---:|:---|:---:|
| 数据更新 | 每日 | 净值数据更新 | 自动 |
| 排名计算 | 双周 | 计算各类别排名 | 自动 |
| 候选筛选 | 双周 | 筛选进入前10%的基金 | 1-2小时 |
| 快速分析 | 月度 | 对观察库基金进行分析 | 4-6小时/只 |
| 深度入库 | 季度 | 对核心库基金深度研究 | 8-12小时/只 |
| 表现跟踪 | 季度 | 已入库基金表现评估 | 4-6小时 |
