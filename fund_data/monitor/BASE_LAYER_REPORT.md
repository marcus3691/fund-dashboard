## 一、系统功能

### 监控类型

| 异常类型 | 说明 | 触发条件 |
|:---|:---|:---|
| **业绩突变** | 业绩暴涨/暴跌 | 单月收益>20%或<-15% |
| **规模异动** | 规模大幅变化 | 规模变化>50% |
| **调仓异常** | 持仓集中度异常 | 前5大占比>60%或<5% |
| **风格漂移** | 投资风格偏离 | 价值型重仓科技股等 |

### 异常分级

| 级别 | 说明 | 响应措施 |
|:---|:---|:---|
| 🔴 **Critical** | 严重异常 | 立即关注，排查原因 |
| 🟠 **High** | 高度异常 | 重点关注，持续跟踪 |
| 🟡 **Medium** | 中度异常 | 定期监控，记录备案 |
| 🟢 **Low** | 轻度异常 | 观察即可 |
