## 八、市场观察与投资建议

### 权益市场数据观察

**估值水平**：当前A股估值处于近十年中性水平，风险溢价处于中性水平。

**资金观察**：
- 海外主动资金连续第2周增持A股约0.91亿美元
- 海外被动资金增持A股港股约7.25亿美元
- 南向资金周度增持港股397.28亿元人民币
- 今年南向累计净买入已达9,192亿元人民币

**ETF观察**（涨幅领先）：
- 稀土ETF、通信ETF、人工智能ETF、云50ETF等，涨幅超74%
- 近一个月：通信ETF、人工智能ETF涨幅超过45%

### 投资管理人观点综述

**权益市场观点**：
- **乐观派**（华泰、平安、富国、中金、南方）：资金流入和结构性机会主导
- **谨慎派**（工银瑞信、人民养老）：强调波动和结构调整

**板块偏好**：
- 工银瑞信：低估值传统行业（银行、资源）
- 平安、国泰、人民养老：科技（AI、硬件）、反内卷主题
- 中金、人民养老、南方：港股补涨（科技互联网）

**债券市场观点**：
- **配置派**（工银瑞信、华泰资产）：利率"上有顶"，债市不悲观
- **谨慎交易派**（人民养老、国泰基金）：对长端债券和转债谨慎

### 投资建议

**权益策略**：
1. 重视权益资产及市场轮动的交易机会
2. 延续"红利+成长"主要配置思路
3. 重视ETF工具运用，提升交易效率
4. 量化策略推荐行业：农业+化工+稀土+券商+A500+AI+港股创新药+港股互联网

**固收策略**：
1. 短期偏防御，控制久期，适度交易
2. 适度增配报价合理的成本类资产
3. 维持信用品中性久期（3-4年）
4. 杠杆率25%左右，降低资金成本
5. 转债偏向防守反击策略

**资产配置建议**：

| 资产类别 | 配置中枢 | 下限 | 上限 |
|:---|:---:|:---:|:---:|
| 权益（扣减套保） | 23% | 0% | 30%（建议提升至35%） |
| 其中：港股通 | - | - | 6%（建议提升至12%） |
| 固收-市值类 | 73% | - | - |
| 其中：可转债 | 0% | 0% | 7% |
| 固收-成本类 | 25% | - | 40%（有报价优势可提升至45%） |
| 流动性资产 | 7% | 5% | 10% |
| 杠杆 | 25% | 15% | 40% |

### 全年收益预测

基于不同情景假设：

|:---|:---:|

## 九、2025年工作成效与存在不足

### 工作成效

| 维度 | 成果 |
|:---|:---|
| **受托监督** | 重新梳理各组合业绩对标，促进良性竞争；启动2025年投资管理人选聘调研 |
| **策略推动** | 推动ETF投资工具应用，截至8月末ETF基金市值9.67亿元，浮盈1.04亿元；四个组合落地量化产品或策略 |
| **风险管理** | 按月开展债权类资产主体排查；向违约存款行提起仲裁 |
| **资产配置** | 完成固定收益组合存量资金调整约6亿元；受托户转入匹配资金4723.76万元 |
| **系统建设** | 完成新一代核心系统信创版单轨切换；投研平台年金投资分析系统上线 |

**量化策略亮点**：
- 国泰组合增配量化投资经理，管理资产在5-8月获得70%+回报，盈利超2900万

### 存在不足

| 方面 | 问题 |
|:---|:---|
| **持仓结构与交易** | 港股投资引导不足，占比提升较慢；债券交易上部分投管人久期节奏表现不佳，利率债亏损 |
| **权益仓位管控** | 开年股债市场波动加剧，为保年度正收益，错失反弹收益；5-7月整体加仓力度与速度不足 |
| **机构管理** | 中金组合投资优化改进缓慢，直至6月推荐量化产品策略，影响组合收益，拖累计划整体 |
