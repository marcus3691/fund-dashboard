## 三、2026年工作推动——组合优化方案

### 面临的三大挑战

1. **路径依赖与能力边界**：投资经理能力圈受限，市场风格切换时业绩波动大
2. **难以克服的人性弱点**：不愿纠正错误、难以长期持有龙头股、年度考核压力下的短期行为
3. **尾部磁吸效应**：组合一旦落入后段，较难翻身

### 优化措施

| 措施 | 进展 |
|:---|:---|
| **分仓策略** | 8个标准含权组合中6个已确认分仓方案和投资经理 |
| **ETF量化轮动** | 其余2个组合已积极配置ETF和稳健量化产品 |
| **每月跟踪** | 将持续跟踪分仓业绩 |

- 万得普通股票型基金指数：4.50%
- **超额收益：+7.94%**

成立以来（2025年9月15日）累计收益：**12.63%**，跑赢指数6.04%

### 3月策略指引（与前期一致）

**市场判断**：A股有韧性，短期震荡

**仓位**：中性，对标年金市场平均水平

**看好板块**：
1. 新能源锂电、电力设备、电池材料
2. 科技上游产业链：存储芯片、光纤、半导体设备、机器人
3. 题材类：商业航天、卫星
4. 高股息板块、资源（煤炭、化工、粮食）

**不看好的板块**：AI应用、军工、消费、创新药

## 五、汇报要点提示

### 成绩亮点
1. 2025年投资收益15.14亿创历史新高
2. 大型银行年金排名2025年第3，近5年累计第1
3. 工银瑞信、国泰稳收组合表现突出（9%+收益率）
4. 专家投资指引策略组合超额收益显著（12.44% vs 4.50%）

### 挑战与应对
1. 投资经理能力圈局限 → **分仓策略**
2. 人性弱点与短期行为 → **ETF量化轮动**
3. 尾部组合翻身难 → **每月跟踪分仓业绩**
