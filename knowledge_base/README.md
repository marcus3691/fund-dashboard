# 研报知识库索引

## 分类说明

| 分类 | 存储内容 | 文件命名格式 |
|:---|:---|:---|
| **宏观策略** | 经济数据、货币政策、市场策略、大势研判 | `YYYYMMDD-机构-标题.md` |
| **行业研究** | 单一行业深度、产业链分析、技术路线 | `YYYYMMDD-机构-行业-核心观点.md` |
| **基金分析** | 基金研究方法、优秀基金案例、AI归因框架 | `YYYYMMDD-标题.md` |
| **ETF研究** | ETF对比、赛道分析、配置建议 | `YYYYMMDD-主题.md` |
| **地缘政治** | 国际冲突、能源安全、供应链风险 | `YYYYMMDD-标题.md` |
| **黄金商品** | 金价分析、通胀逻辑、避险策略 | `YYYYMMDD-标题.md` |
| **政策解读** | 五年规划、产业政策、监管变化 | `YYYYMMDD-政策名-解读.md` |
| **个股公司** | 公司深度、财报分析、调研纪要 | `YYYYMMDD-公司代码-标题.md` |
| **错误记录** | 研究错误、数据问题、教训总结 | `YYYYMMDD-错误简述.md` |
| **待验证观点** | 需要跟踪验证的预测、假设 | `YYYYMMDD-观点简述.md` |

## 使用方式

### 存入新研报
1. 将原文核心观点摘要整理
2. 按分类存入对应文件夹
3. 文件名包含日期和核心关键词
4. 文末标注：来源链接、分析师、记录时间

### 检索使用
- 按分类浏览相关主题
- 按日期查看历史观点演变
- 跨分类关联（如政策→行业→基金）
