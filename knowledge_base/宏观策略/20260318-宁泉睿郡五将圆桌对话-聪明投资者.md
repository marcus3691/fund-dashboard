## 二、市场判断与估值观点

### 1. 杨东：估值是感知市场热度最重要的温度计

**核心观点：**
- 当前市场有"相当热度"
- 但整体周期已碎化为行业周期和个股周期，同涨同跌的可能性越来越小
- 最重要的感知指标是**估值**
- 观察信号：新基金发行、两融数据、市场交投数据等

**结构性判断：**
- 一些板块已经走过周期高位
- 一些板块还在底部或周期未开始
- 近期化工、周期股的上涨就是行业周期在动

### 2. 杜昌勇：权益仍是性价比不错的资产

**大类资产配置观点：**

| 资产类别 | 判断 | 理由 |
|:---|:---|:---|
| 地产 | 大周期大概率已过 | 人口结构变化，2020年是转折点 |
| 债券 | 回报有限 | 10Y/20Y国债收益率<2%，难抵御通胀 |
| **权益** | **性价比不错** | 自下而上能找到估值合理的优秀公司，一批公司分红率>4-5% |
| 大宗商品 | 波动大，警惕 | 黄金等已大幅上涨，"天然对短时间内大幅上涨资产保持警惕" |

**黄金观点：**
- 通过黄金股而非直接配置黄金
- 几年前曾参与黄金龙头定增并长期持有
- 但在目标价离场后，股价又涨了一倍多（偏左侧上车，有时提早下车）
- **当前判断**：面对过去三年巨大涨幅，不再追加配置

### 3. 董承非：今年市场可用"混沌"二字形容

**与2024年的对比：**
- 2024年值得乐观（ hence 橙色封面图）
- 2025年后市场热度能否延续？可能被事件打断？
- **结论**：混沌、不清晰，没有特别明确的判断

**AI投资两阶段思考：**
1. **基建阶段**（数据中心）：商业模式不好，不是时间的朋友，睿郡错过
2. **应用阶段**：终于看到AI落地迹象，"看到了曙光，蛮激动的"

**对软件公司的重新思考：**
- AI正在颠覆底层IT架构，中国只晚了半步入场
- 追赶速度快，对很多公司可能是机会
- 中国实力提升背景下，AI相关产业链有机会在全球赚到钱

**风险提示：**
- 2000年互联网泡沫中间被打断，调整了两年
- 这一轮会不会也出现类似打断因素？每天都在问自己，暂无答案

## 三、投资哲学与方法论

### 1. 安全边际（杨东）

**本质理解：**
- 安全边际是一种**思维方式**，不是僵化公式
- 强调：判断出错或小概率事件发生时，能够少亏甚至不亏
- 简单理解：价值10块的股票，6-7块买入，折扣就是安全边际

**灵活应用：**
- 熟悉的股票，安全边际可放低
- 复杂行业或信心不足，安全边际要多留
- 仓位低时，个股安全边际可放宽
- 市场整体低迷时，安全边际标准可放大
- 市场缺乏高安全边际标的且仓位低时，标准需适当放低

**核心原则：**
- **风险控制能力就是投资能力本身**，不是额外能力
- 最终收益 = 赚钱部分 - 亏钱部分
- 尽量减少亏损次数和金额，同样能实现好收益
- 投资不能像赌博，保持平常心

### 2. 重大决策的形成（王晓明）

**三点体会：**

1. **对市场保持谦卑**
   - 始终认为自己可能会犯错
   - 承认自己可能犯错，但犯错不致命

2. **只有一条命**
   - "做资产管理这个行业，只有一条命，不能在任何一次重要决定里把这条命赔掉"
   - 犯错后不能置于无法挽回的境地

3. **重大决定是长时间持续思考的结果**
   - 没有任何重大决定是突然产生的
   - 案例：杨东2018年"5·31"光伏新政后持续买入
   - 背后是往前推很多年对光伏行业的持续跟踪、深度研究（从兴全时期就订阅专业杂志，研究员持续关注）
   - 2018年的决定，是长达十年甚至更久积累的结果

4. **机会来临时要敢于下注**
   - "谁家都有吃饺子的时候，轮到你的时候，得吃饱、吃足"
   - 当真正重要的时刻来临时，要敢于把胜利发挥到极致

**当前判断：**
- 还没到要做重大、决定性抉择的时候
- 9·24之后很多股票涨幅不小且缺少基本面支撑
- 目前还是中间阶段

### 3. 转债投资（薛大威）

**睿郡转债定位：**
- 非常看重"不对称的风险收益比"——下跌空间有限，向上有弹性
- 核心要求：跌的时候少跌甚至不跌，涨的时候跟得上市场

**当前转债市场判断：**
- 价格中位数一度到140，平价100出头
- 隐含正股至少30%以上涨幅预期，估值在历史高位
- 已做出决策：整体仓位降下来

**寻找转债阿尔法的方法：**
1. **正股研究**：围绕行业和公司基本面
2. **风险收益不对称品种**：向下空间有限、向上有弹性（符合转债初衷）
3. **市场错误定价**：如债务压力下转债定价低估，深入研究企业偿债能力和转股可能性
4. **拒绝高价高溢价**：高价+高溢价状态的标的通常放弃

### 4. 主动投资vs AI时代（董承非、薛大威）

**薛大威：**
- 主动管理面临挑战是常态（十几年前有信托，现在有量化）
- 基于基本面驱动、寻找有价值成长性企业的主动投资始终有立足之地
- AI赋能后，市场对短期催化的定价效率越来越高
- **不能去拼短期事件反应速度**（比不过量化）

**构建主动投资竞争力的两点：**
1. 对企业看得更前瞻、更深入、更长远，耐心等待安全边际价格，长期持有挖掘价值
2. 具备**逆向思维**能力——短期看全是风险，拉长到两三年反而可能是机会

**董承非：**
- "既然拼快拼不过，那就跟它拼慢"
- 主动把节奏降下来
- AI强项：对已经发生的信息处理快、总结好
- AI短板：对未来做判断、做前瞻，做不到那么深
- **我们要做的：看得更深一点、更远一点**
- 研究员要求：用AI工具，但和市场保持一定距离，结论才有前瞻性、有深度
- "时间会站在我们这边，慢慢验证我们的判断"

## 四、选股与行业判断

### 1. 杨东：新能源投资的复盘与反思

**2018年的判断：**
- 第一批产品2018年成立，市场环境不好
- 对新能源的看好是保持信心的重要支撑
- 当时认为可选标的有限（偏保守）

**回头看的问题：**
- 在这么大的产业浪潮里，**所有的船其实都会浮起来**
- 质地较差或二三线企业，涨幅反而比一线龙头还大
- 过于谨慎，要求太高（成本最优、竞争优势最强），错失很多牛股

**当前选股标准：**
- 估值很重要
- 安全边际很重要
- 行业竞争优势很重要
- 行业处于残酷洗牌阶段，体质不好的公司可能直接出局

**对泡沫的思考：**
- 预判到新能源一定会有泡沫，但对泡沫的容忍度还是太小
- "看对的时候要赚足"——这方面犯过错，看对方向但很多收益没抓住
- 未来遇到大浪潮时，会适当放宽选择范围

**放宽安全边际的前提：**
- 除非能明确预判一家企业未来有10倍以上增长空间
- 不认同"看好长远前景就可以不用看估值"
- "如果它真有这么大空间，却还以当前价格交易，是不是意味着市场里其他人都错了？"
- 对估值的把握，一定是非常重要的

### 2. 王晓明：选赛道与企业家精神

**选赛道（长坡厚雪）：**
- 坡足够长非常关键
- 有些行业持续时间长（如石油），有些行业璀璨时间很短
- 传统行业虽然不性感，但估值低，长期持有回报惊人
- 性感行业波动大、有毁灭性风险，可能三五年就被淘汰
- 要随着时代变化，选择**足够长、能持续发挥、商业模式简单、赚钱容易**的行业

**企业家精神：**
- 所有优秀企业最终都离不开核心灵魂人物——企业家
- 企业家精神（极具灵性、坚韧、持之以恒）是观察企业非常重要的一环
- 和优秀企业家同行，投资成功概率会提升很多

## 六、与年金投资策略的关联与启发

### 1. 估值与仓位管理的呼应

| 宁泉睿郡观点 | 年金策略指引 |
|:---|:---|
| 估值是感知热度的温度计 | 控制回撤为主，稳中略降 |
| 对短期大幅上涨资产保持警惕 | 3月调整前已提示控制仓位 |
| 转债估值在历史高位，已减仓 | 限制转债敞口5%上限 |

**启发：** 年金策略在防御性判断上与顶级私募一致，都是基于估值和安全边际的考虑。

### 2. "混沌"与策略模糊的正确

- 董承非说今年"混沌"，没有明确判断
- 年金指引也强调"控制回撤"而非"主动进攻"
- **启发：** 在方向不清晰时，宁可错过不做错，年金投资尤其如此（不能冒无法翻身之险）

### 3. 分仓管理与风格互补

| 睿郡/宁泉 | 国泰年金 |
|:---|:---|
| 多位合伙人风格互补 | 三位经理分仓管理 |
| 杨东：估值/左侧布局 | 施钰：选股/长期持有 |
| 董承非：混沌/看得远 | 叶烽：平衡/A股+港股 |
| 薛大威：转债/不对称收益 | 曾辉：交易/港股波段 |

**启发：** 分仓模式的本质是**多元风格覆盖不同市场环境**，私募通过合伙人实现，年金通过分仓经理实现，底层逻辑一致。

### 4. 主动投资的"慢"哲学

- 董承非："拼快拼不过，那就拼慢，看得更深更远"
- 国泰施钰：低换手、长期持有，价差占比仅20%
- **启发：** 年金投资同样不能追求短期交易，选股型经理（如施钰）的长期持有能力是核心阿尔法来源。

### 5. 安全边际与回撤控制

- 杨东反复强调"不能遭受毁灭性打击"
- 王晓明"只有一条命"
- 年金目标：平衡绝对收益、相对收益、正回报，不能有太大下行波动
- **启发：** 两者在风险控制哲学上高度一致，年金的分仓策略正是为了分散"单条命"的风险。

### 6. 黄金判断的验证

- 杜昌勇：面对黄金三年巨大涨幅，不再追加配置
- 年金3月策略指引：黄金作为防御性配置，但未主动加仓
- **启发：** 对过热资产的警惕是专业投资者的共性。
