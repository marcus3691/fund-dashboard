## 二、市场展望及投资策略

### （一）宏观经济与流动性

| 维度 | 判断 |
|:---|:---|
| 外部环境 | 冲击趋缓 |
| 政策方向 | 从稳增长向**调结构**倾斜 |
| 经济增长 | 回归内生增长，趋于平淡 |
| 流动性 | 充裕，居民储蓄向资本市场转移 |
| 投资者情绪 | 多数持低仓位观望 |
| 经济周期 | 底部逐步显现 |
| 后续关注 | 资源品估值修复后，**业绩驱动型机会** |
| 风险点 | 潜在外部风险（美股调整→科技股冲击） |

### （二）权益市场策略

**整体判断**：看好全年机会，但**波动加剧**，可能出现类似**2015年的结构性行情**

**看好方向**：
1. **优质公司估值修复**（价值风格）
2. **成长赛道**（AI、创新药等）
3. **周期板块**（有色、工程机械等）
4. **港股**（积极捕捉交易和配置机会）

### （三）固收市场策略

**整体判断**：2026年仍为**震荡行情**

**操作原则**：以稳为主，绝对收益基础上适度增强

**重点关注**：
| 资产类别 | 策略 |
|:---|:---|
| **"固收+"资产** | 转债、二级债基等，作为弹性来源 |
| 市值类资产短端 | 受益于资金宽松，套息策略占优 |
| 市值类资产长端 | 受供给压制，逢脉冲式冲击配置 |
| 久期管理 | 波段策略，灵活操作 |

## 三、沟通交流环节

### （一）年金管理模式新范式

#### 当前挑战与痛点

| 痛点 | 表现 |
|:---|:---|
| **路径依赖与能力边界** | 投资经理能力圈所限，风格切换时业绩波动大 |
| **人性弱点** | 不愿纠正错误、难以长期持有龙头股、年度考核压力下的短期行为 |
| **尾部磁吸效应** | 业绩落入后段后，客户体验和内部压力下难以翻身 |

#### 解决方案：产品化配置 + 量化赋能

**1. 产品化配置模式**

| 维度 | 内容 |
|:---|:---|
| 精力分配 | 70%核心策略（选股）+ 30%配置互补产品 |
| 配置标的 | 公司内部优秀的、与自身策略互补的基金（ETF、其他基金经理专户） |
| 目标业绩 | 锚定长期稳健的前1/3分位 |
| 核心优势 | 提升客户体验、降低波动、收益来源多样化 |
| 试点状态 | 已在养老金产品中试点成功，正向年金推广 |

**2. 量化与系统化工具赋能**

| 工具 | 功能 |
|:---|:---|
| ETF投资决策系统 | 整合多因子模型、行业轮动观点、券商共识 |
| AI应用 | 研报解读、因子挖掘、绩效分析、风险预警 |
| 目标 | 从"经验驱动"到"数据与系统驱动"的工业化协作 |

### （二）重点关注行业讨论

| 行业 | 观点 |
|:---|:---|
| **AI硬件**（光模块、PCB） | 业绩确定、估值合理，但股价已含预期，短期可能横盘震荡，关注中国供应链替代机会 |
| **AI应用** | 面临挑战，大模型厂商挤压中间层，大规模爆发需等技术门槛降低 |
| **新能源** | 核心问题仍是产能过剩，需等待供需改善和新需求 |

## 四、总结

### 核心要点

1. **业绩评价**：债券端优秀，权益端中等，整体中等偏上
2. **市场判断**：看好全年但波动加剧，类似2015年结构性行情
3. **权益方向**：估值修复 + AI/创新药成长 + 周期板块
4. **固收策略**：以稳为主，重视"固收+"弹性
5. **模式创新**：产品化配置（70%选股+30%配置）+ 量化工具赋能

### 与理事会其他管理人的对比（待后续纪要补充）

**富国特点**：
- 固收能力强（纯债前12%分位）
- 权益偏保守（跑输股票型基金指数）
- 正在推广产品化配置新模式
- 积极应用AI和量化工具
