## 二、市场展望及投资策略

### （一）宏观基本面：有望企稳回升

| 指标 | 判断 |
|:---|:---|
| PPI/CPI | 企稳回升 |
| 工业企业利润 | 转正 |
| 内需回暖 | 可能需待下半年 |
| 2026年定位 | **通缩转折之年** |
| 企业盈利 | 有望持续改善 |
| 房地产 | 预计2026年下半年-2027年上半年企稳 |

### （二）市场展望：结构性行情可期但波动加剧

**整体判断**：
- 看好全年权益市场机会
- **波动加剧**，可能出现类似**2015年结构性行情**
- **指数目标**：4400-4500点

**资金面**：
- 美元贬值背景下外资流入可期
- 国内居民资金通过理财、ETF持续入市

**主要风险**：
1. 美股估值高位可能调整→传导至A股科技股
2. A股部分题材股泡沫化（如AI概念）
3. "国家队"调控行为影响市场节奏

### （三）投资策略：均衡布局，保持警惕

**1. 权益策略**

**仓位管理**：
- 年初市场调整时积极加仓
- 当前权益仓位已**接近上限**
- 后续计划波动操作，等待下一次加仓机会

**行业配置思路（均衡为主）**：

| 主线 | 方向 | 逻辑 |
|:---|:---|:---|
| **价值防守** | 房地产、消费 | 底部安全边际高，向下空间有限，经济复苏后具备弹性 |
| **资源品** | 有色金属 | 地缘政治紧张、军备开支增加→供需紧张→价格上涨压力 |
| **科技成长** | 半导体、算力 | 有实际业绩支撑，对纯炒概念题材持谨慎态度 |
| **港股机会** | 互联网、红利股 | 外资回流+美元贬值背景下的配置价值 |

**2. 固收策略**

| 维度 | 策略 |
|:---|:---|
| 核心 | 以票息防御为主 |
| 杠杆 | 维持审慎杠杆 |
| 久期 | 保持低久期配置 |
| 信用债 | 关注3-5年期配置价值 |
| 利率债 | 等待超调机会 |

## 三、沟通交流环节

### （一）"产品化+量化"的投资新范式

**我方（理事会）提出的转型要求**：

| 维度 | 内容 |
|:---|:---|
| 核心观点 | 从传统"炒股"转向更系统化的投资范式 |
| 背景 | 个人选股投资难以持续，弊端很多 |
| 模式 | **产品化+量化** |
| 产品化 | 配置ETF、指数增强基金、量化产品等工具型产品把握行业贝塔 |
| 量化 | 建立金融工程团队，用量化模型穿透管理产品，分析行业/主题暴露，动态轮动 |

**宏观视角转变**：
- 打破旧有框架
- 认识中国制造业全球化带来的巨大机遇
- 未来龙头公司将呈现"赢家通吃"格局
- 投资需要更宏观、更系统的视角

### （二）更灵活的组合权益仓位上限

**平安养老诉求**：
- 希望放宽组合权益仓位上限至**35%**
- 理由：看好市场时更充分配置资产，实现公平竞争

**我方回应**：
- 表示理解
- 上调权益仓位上限基于**历史业绩表现**
- 管理人可提出正式申请
- 我方会积极考虑并提请呈批
