### 二、地缘冲突与黄金有色板块
**核心矛盾**: 伊朗冲突升级(2月底至今)，黄金反而下跌

**传导链条**:
```
地缘冲突 → 油价↑ → 通胀↑ → 美联储推迟降息 → 实际利率↑ → 黄金↓
         → 中东国家抛售黄金换流动性 → 金价进一步承压
```

**关键洞察**:
- 黄金不是避险资产，而是"流动性血包"
- 流动性冲击 > 避险需求
- 等待信号：美债利率企稳 = 流动性冲击结束

**年金策略修正**:
- 黄金：从"防御配置"调整为"观望"
- 有色分化：原油/煤炭/煤化工持有，黄金股避开
