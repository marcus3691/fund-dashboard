# 黄金分析学习笔记

**来源**: 政信三公子《黄金一周暴跌超3%：牛市终结还是深蹲起跳？》
**时间**: 2026-03-14

---

## 一、文章核心观点

### 本周金价走势
- 从5200美元跌至5000美元关口，单周跌幅超3%
- 2026年以来最惨一周，2024年12月以来最大周度回调

### 下跌逻辑（宏观传导链）

```
原油暴涨(75→100美元, +30%)
    ↓
通胀预期升温(汽油+17%, CPI能源+7%)
    ↓
降息预期降温(6月降息概率65%→39%)
    ↓
美债收益率↑ + 美元指数↑
    ↓
金价承压(黄金是无息资产, 与实际利率负相关)
```

### 微观结构放大跌幅
1. **获利了结**: 2月春节提前透支避险预期，战事爆发后集中涌出
2. **流动性挤兑**: 油价暴涨引发风险资产普跌，黄金成"流动性血包"被抛售换现金
3. **技术性破位**: 跌破50日均线，触发程序化止损，多杀多

## 四、对ETF信号系统的启发

### 1. 宏观传导链分析框架

文章展示了清晰的传导逻辑：
```
地缘事件 → 商品价格 → 通胀预期 → 货币政策 → 资产价格
```

**应用到ETF信号:**
- 地缘冲突信号不能只盯着冲突本身
- 要跟踪: 油价 → 通胀预期 → 美联储政策 → 各ETF表现
- 建立"传导链监控"机制

### 2. 流动性挤兑风险识别

文章指出：黄金被抛售是因为"流动性最好、前期盈利丰厚"

**启发:**
- 极端行情下，流动性好的资产反而先跌（被抛售换现金）
- ETF信号应增加"流动性挤兑"风险标记
- 高波动时期，减仓流动性好的ETF可能被动触发

### 3. 机构观点聚合分析

文章汇总了5家顶级机构观点，提取共识和分歧：
- **共识**: 长期看多（央行购金+降息周期）
- **分歧**: 短期谨慎 vs 乐观
- **风险提示**: 地缘引发的涨势往往难以持久

**应用到信号系统:**
- 增加"机构观点聚合"模块
- 提取共识作为中长期信号，分歧作为短期波动提示

### 4. 情景概率分析框架

文章给出4种情景及概率分布：
- 基准45% + 偏弱30% + 危机15% + 升级10% = 100%

**启发:**
- 我们的信号也可以用概率思维表达
- 不是"涨或跌"，而是"基准情景概率X%，极端情景概率Y%"
- 对应不同的仓位配置

### 5. 技术面与基本面结合

文章结合了：
- 基本面: 地缘、美联储、通胀数据
- 技术面: 50日均线、5000美元关口、程序化止损

**应用到信号:**
- 增加技术面验证（均线、支撑位、成交量）
- 基本面信号+技术面确认=更高胜率

### 6. 分批建仓策略

文章给出的中长线操作建议是分3批建仓：
- 不是一次性梭哈，而是根据价格梯度分批

**启发:**
- 我们的信号应该包含"分批建仓"指引
- 比如: 第一批5100(20%)、第二批5000(30%)、第三批4850(30%)
- 避免择时压力，平滑成本

## 五、可迁移的分析框架

### 黄金分析框架 → ETF分析框架

| 黄金维度 | ETF对应维度 |
|:---|:---|
| 央行购金 | 北向资金流向、机构持仓变化 |
| 降息预期 | 行业景气度、政策支持力度 |
| 地缘风险 | 产业链风险（原材料、出口） |
| 通胀传导 | 成本传导（上游→下游） |
| 流动性挤兑 | 赎回压力、成交量萎缩 |
| 技术面 | 均线、支撑位、MACD |

### 建议增加的信号类型

1. **宏观传导链信号**
   - 监测: 商品价格 → 通胀 → 政策 → ETF表现
   - 应用: 提前预判ETF受影响方向和幅度

2. **流动性风险信号**
   - 监测: 市场波动率、赎回压力、成交量异常
   - 应用: 高波动时期调整仓位，避免流动性挤兑

3. **机构共识信号**
   - 监测: 顶级机构研报观点聚合
   - 应用: 提取中长期共识，识别短期分歧

4. **技术面确认信号**
   - 监测: 均线、支撑位/压力位、成交量
   - 应用: 基本面信号+技术面确认，提高胜率

5. **分批建仓信号**
   - 监测: 价格梯度、估值分位
   - 应用: 不是一次性调仓，而是分3-5批平滑成本
