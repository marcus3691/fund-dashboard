# 基金分析模板

## 单产品深度分析

### 数据准备清单
- [ ] 基金代码
- [ ] 2020年底以来季度前十大重仓股
- [ ] 月度净值数据
- [ ] 基金经理任职时间

### AI分析提示词

#### 第一步：投资框架分析
```
上面是基金[基金代码]从2020年底以来每个季度的重仓股和月度净值数据。

请分析：
1. 该基金经理的投资框架是什么？
2. 投资特点和风格特征？
3. 重点关注的指标有哪些？
4. 调仓逻辑和持股周期？

请用一句话概括该基金经理的投资框架。
```

#### 第二步：能力圈分析
```
基于同样的数据，请分析：

1. 能力圈强势领域（行业、风格、市场阶段）
2. 能力圈弱势领域
3. 和同风格、同波动水平的基金经理相比，投资能力如何？
4. 适合什么市场环境？不适合什么环境？
```

#### 第三步：投资建议
```
基于以上分析，请给出：

1. 如何投资这位基金经理？（买入时机、持有周期、卖出信号）
2. 配置比例建议
3. 风险控制要点
4. 基金经理需要扩展哪些能力？需要保留哪些能力？
```

## 赛道产品对比分析

### 对比维度
1. **投资风格**：价值/成长、大盘/小盘
2. **选股逻辑**：自下而上/自上而下、质量/景气度
3. **换手率**：持股周期、调仓频率
4. **业绩归因**：alpha来源、beta暴露
5. **风险控制**：最大回撤、波动率控制

### 提示词模板
```
以下是两位基金经理的重仓股和净值数据：

基金经理A：[数据]
基金经理B：[数据]

请对比分析：
1. 两人的投资风格差异
2. 选股逻辑有什么不同
3. 各自的优劣势
4. 适合什么样的投资者
5. 在当前市场环境下，更看好谁？为什么？
```

## Smartbeta产品分析

### 分析要点
1. **指数编制方法**：加权方式、选股规则
2. **因子暴露**：价值/成长/质量/动量等因子
3. **行业分布**：集中度、分散度
4. **历史表现**：不同市场环境下的表现
5. **当前估值**：PE、PB分位数

### 提示词模板
```
以下是[指数名称]的编制规则和成分股数据：

请分析：
1. 该指数的编制逻辑和特点
2. 主要暴露在哪些因子上
3. 适合什么市场环境
4. 当前估值水平如何
5. 投资建议
```

## 数据获取方式

### Tushare Pro
```python
import tushare as ts
pro = ts.pro_api('your_token')

# 基金净值
df = pro.fund_nav(ts_code='基金代码')

# 基金持仓
df = pro.fund_portfolio(ts_code='基金代码')

# 基金经理
df = pro.fund_manager(ts_code='基金代码')
```

### 数据来源
- Tushare Pro
- Wind（如有权限）
- 天天基金网
- 基金季报/年报

## 输出报告结构

### 1. 执行摘要
- 一句话投资框架
- 核心结论（买/不买/观望）

### 2. 基金经理画像
- 投资风格
- 能力圈
- 优势/劣势

### 3. 业绩分析
- 收益表现
- 风险控制
- 同风格排名

### 4. 投资建议
- 配置比例
- 买入时机
- 风险提示

### 5. 跟踪要点
- 需要关注的变化
- 调仓信号
- 退出条件
