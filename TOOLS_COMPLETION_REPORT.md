## 一、已完成工具清单

### 1. 研报/纪要AI解析流水线 ✅
**文件**: `scripts/report_parser.py`

**功能**:
- 自动解析PDF研报和会议纪要
- 提取文本层或转换图片为可处理格式
- 自动识别核心观点（宏观、行业推荐、风险提示、配置建议）
- 输出结构化JSON格式，便于后续分析

**使用方法**:
```bash
python3 scripts/report_parser.py /path/to/report.pdf
```

**输出示例**:
- 解析状态（页数、是否有文本层）
- 核心观点分类提取
- 图片资源保存
- 结构化JSON文件

### 2. 飞书指令快速查询工具 ✅
**文件**: `scripts/fund_query_handler.py`

**功能**:
- 处理自然语言查询指令
- 快速分析单只/多只基金持仓与会议纪要方向匹配度
- 支持交互模式和命令行模式

**支持查询类型**:
| 查询示例 | 功能 |
|:---|:---|
| "分析 025476.OF 匹配能源安全方向" | 单基金方向匹配分析 |
| "TOP10基金匹配能源安全方向" | 批量基金匹配排名 |
| "哪些基金符合周期股方向" | 按行业筛选基金 |
| "TOP20基金持仓" | 查看TOP基金列表 |

**使用方法**:
```bash
# 单条查询
python3 scripts/fund_query_handler.py "TOP10基金匹配能源安全方向"

# 交互模式
python3 scripts/fund_query_handler.py -i
```

- 发现多个一致性方向：煤炭(3来源一致看好)、半导体(2来源一致看好)

